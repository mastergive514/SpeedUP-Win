# Speed UP
Speed up is a free python file, can speed up your pc!



# Instalation
**Download files in your pc**

```$ git clone https://github.com/mastergive514/SpeedUP-Win.git```

**go to Directory of files**

```$ cd SpeedUP-Win```

**Run SpeedUP**

```$ Speedup.exe```
